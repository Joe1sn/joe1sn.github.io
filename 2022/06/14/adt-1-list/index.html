sequence VS. link

Time Performance

  • Data Access

    ​ The one of specialties is random access structure.If you randomly access data and the time complexity is O(1)O(1)O(1)
    ​ But the link list don’t have this specialty,so it will take more time to find the data, and the time complexity is O(n)O(n)O(n)

  • Insert and Delete

    ​ The link list once found the postion to operate, just need change the pointer to finished job.But sequence list need move data,and it’s complexity is O(n)O(n)O(n)

  • Conclude

    If we need frequently to insert or delete, link list is better
    If we need frequently to access data, sequence list is better
