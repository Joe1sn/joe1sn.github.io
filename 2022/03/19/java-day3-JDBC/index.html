JDBC概念

**概念：**Java DataBase Connectivity ，Java数据库连接

本质（类似Django的ORM）:The sun公司定义的一套操作所有关系型数据库的规则（接口）。各个厂家根据接口去实现。我们可以是有这套接口编程，真正的驱动时jar包中的实现类

快速入门

简单步骤

  1. 导入驱动jar包
  2. 注册驱动
  3. 数据库连接对象Connection
  4. 定义sql
  5. 获取执行SQL语句的Statement对象
  6. 执行sql，接受返回结果
  7. 处理结果
  8. 释放资源
